# PathPilot AI Architecture

This project uses Retrieval-Augmented Generation (RAG) to recommend jobs.

Workflow:

User Input → Knowledge Retrieval → LLM (Gemini) → Answer

Explanation:

1. User enters their skills.
2. Knowledge Retrieval searches the job dataset.
3. Gemini LLM analyzes the retrieved data.
4. The chatbot generates job recommendations.
