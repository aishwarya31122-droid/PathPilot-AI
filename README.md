# PathPilot AI – Job Recommendation Chatbot

PathPilot AI is an AI-powered chatbot that recommends jobs based on user skills.

## Features
- Job recommendation using AI
- Uses Kaggle job dataset
- Built with Dify and Gemini

## Workflow
User Input → Knowledge Retrieval → LLM → Answer

## Example Query
"I know Python and Data Analysis. Suggest jobs for me."
